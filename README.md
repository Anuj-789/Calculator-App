# Calculator-App
<h3>About</h3>

A modern and interactive Calculator App designed for quick and easy arithmetic calculations. It allows users to perform basic operations as well as some advanced features, all within a clean and user-friendly interface. Perfect for students, developers, or anyone needing a reliable calculator in the browser.

<h3>Features</h3>

Perform basic arithmetic operations: addition, subtraction, multiplication, division

Decimal calculations supported

Clear (C) and backspace functionality for easy corrections

Keyboard input support for faster calculation

Responsive and mobile-friendly design

Lightweight and fast, runs directly in the browser

Optional: Dark mode toggle for eye comfort

Can be extended for scientific operations like square root, percentages, or memory functions

<h3>Installation</h3>

Clone the repository:

git clone https://github.com/yourusername/calculator-app.git


Open index.html in your browser.

<h3>Usage</h3>

Click the number and operation buttons to input your calculation

Press = to get the result

Press C to clear or use backspace for corrections

Use keyboard numbers and operators for convenience
